Question-1: What are the different ways to select an element in the DOM?

DOM element select করার জন্য আমরা `getElementById()`, `querySelector()`, `getElementsByClassName()` ইত্যাদি ব্যবহার করি।

Question-2: What is the difference between innerHTML, innerText, and textContent?

`innerHTML` HTML parse করে, `innerText` visible text দেয়, আর `textContent` সব text return করে।

Question-3: What is event delegation in the DOM?

Parent element এ event listener set করে তার child elements এর event handle করাকে event delegation বলে।

Question-4: What is event bubbling in the DOM?

Event bubbling মানে, child থেকে parent element পর্যন্ত event propagate হওয়া।
